# BShop
安卓开发框架

这个是android develp的一个框架，我对他的维护目前准备分两个阶段<br> 
* 第一阶段：完成整个工程架构的搭建，主要是闪屏、首页切换，后面会加一个带左边侧滑的首页<br> 
* 第二阶段：会添加上一些解决方案，例如我研究好久的首页的多类型条目的实现<br> 

目的：督促自己的学习，给大加提供一个拿来就能用的安卓框架，不用在开发前期做多余的准备工作，直接上手开发<br>
